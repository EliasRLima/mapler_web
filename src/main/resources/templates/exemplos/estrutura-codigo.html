Estrutura do Código

A Estrutura Básica de um código Portugol consiste em 3 áreas: variaveis, inicio e fim.

  • variaveis: declaração de todas as variáveis e módulos;
  • inicio: marca o início do algoritmo;
  • fim: marca o fim do algoritmo.
            // Estrutura Básica:

            variaveis

                // Nesta área declaramos as variáveis

                // utilizadas no algoritmo

                var_nome: cadeia;

            inicio
            // A partir daqui começa o algoritmo

                escrever "Digite seu nome:";

                ler var_nome;

                escrever "Olá ", var_nome;

            fim // Esta instrução indica o fim do algoritmo
